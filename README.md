## Usage
```
usage: mastermind.py [-h] [--rounds ROUNDS] [--noninteractive]

optional arguments:
  -h, --help        show this help message and exit
  --rounds ROUNDS   number of rounds (must be even)
  --noninteractive  machine vs. machine
```
